# Titanic Exploratory Data Analysis (EDA)

## Objective
To analyze the Titanic dataset and identify factors affecting passenger survival.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## Analysis Performed
- Statistical Summary
- Missing Value Analysis
- Correlation Heatmap
- Gender vs Survival Analysis
- Passenger Class vs Survival Analysis
- Age Distribution Analysis

## Key Findings
- Female passengers had higher survival rates.
- First-class passengers survived more frequently.
- Most passengers were between 20 and 40 years old.
- Fare and Passenger Class influenced survival.

## Conclusion
EDA helped uncover important insights and relationships in the Titanic dataset through statistical analysis and visualizations.
